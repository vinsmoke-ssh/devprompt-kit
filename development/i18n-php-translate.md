# PHP Language File Translation Prompt

## Description
Translate English text inside PHP language array files into Indonesian. Only translate the text value on the right side of the "=" sign without modifying keys, structure, quotes, or semicolons.

## Prompt
Saya ingin anda translate script bahasa Inggris ke bahasa Indonesia pada file bahasa PHP.  
Formatnya sebagai berikut: '$lang['view_details'] = 'View Details';'

Jangan ubah ke key array ('$lang['...']'), cukup terjemahkan teks yang ada di sebelah kanan tanda '=' tanpa mengubah struktur, tanda kutip, atau titik koma.

### Task:
- Kembalikan hasil dalam satu blok kode PHP.
- Pertahankan urutan baris dan format asli.
- Hanya terjemahkan bagian teks Inggris yang berada dalam tanda kutip setelah tanda '='.
- Jangan menambah baris baru atau menghapus baris.
