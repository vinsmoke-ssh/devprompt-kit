# CodeIgniter 3 Project Planning Prompt

## Description
Prompt ini digunakan untuk membantu merencanakan project CodeIgniter 3 secara rapi sebelum mulai coding, khususnya untuk developer pemula yang ingin membangun sistem CRUD berbasis admin panel.

## Prompt
<pre>
Saya seorang [LEVEL DEVELOPER] dalam pengembangan web dan ingin membuat sebuah project menggunakan CodeIgniter 3.

Saya akan membuat sistem berbasis web bernama "[NAMA SISTEM]", yaitu sistem [DESKRIPSI SISTEM].

Fitur MVP yang direncanakan:
[MASUKKAN FITUR]

Tujuan utama saat ini:
[MASUKKAN TUJUAN]

Teknologi yang digunakan:
[MASUKKAN TEKNOLOGI YANG DIGUNAKAN]

Saya ingin Anda membantu saya dengan panduan langkah demi langkah, khususnya:
1. Checklist persiapan sebelum mulai coding (misalnya: perencanaan database, penentuan modul, penamaan folder, dll)
2. Rekomendasi struktur folder & file CodeIgniter 3 yang cocok untuk sistem CRUD berbasis Admin Panel
3. Tips praktik terbaik (best practice) yang mudah dipahami, tetapi tidak salah secara konsep

Gunakan bahasa sederhana, penjelasan bertahap, dan anggap saya baru belajar tapi ingin serius membangun sistem yang rapi.
</pre>
